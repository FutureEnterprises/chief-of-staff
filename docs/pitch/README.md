## The honesty bar

These documents are written under three constraints. Every revision
should be read against them.

1. **Founder-led, not team-led.** Iman Schrock is the sole employee.
   Technical cofounder + clinical advisor + first engineer are
   recruiting targets, not announced hires. Any version of these docs
   that fabricates a team violates the bar.
2. **Pre-revenue.** Stripe is wired; the first paying-user cohort is
   the Tranche 1 milestone. No revenue claims, no "n=187" cohort
   numbers, no fabricated retention figures. The infrastructure is
   real; the revenue is the next thing to manufacture.
3. **No fabricated proof points.** No customer testimonials, no logo
   walls, no advisor names until a credible name confirms. The
   `/advisors` surface returns when there is something true to put on
   it.

If any future edit to these documents introduces a number, a name, or
a logo that fails one of those tests — **revert the edit**. The
founder would rather close a smaller round on the truth than a
larger one on a fiction.
