# Slide 2 — The Insight

# Every AI before COYL wraps around external reality.

# COYL is the first one wrapped around behavioral reality.

Documents. Code. Language. Every foundation-model breakthrough since
GPT-3 has been a wrapper around external artifacts — text the user
produced or external data the user pointed at. The model reasons,
the user reads the answer, the user takes the action. The AI is
downstream of the moment.

COYL is the first AI infrastructure layer that is **upstream of the
moment**. It reads HRV, accelerometer, screen-state, location,
calendar — the passive behavioral signal stack a human emits without
typing anything — and fires inside the three-second window between
intention and behavior. The user does not ask. The AI proposes.

The category jump is from **smart journal** to **thermostat**. A
journal reads what already happened. A thermostat reads the room and
acts before the room gets uncomfortable. Every existing AI is a
journal. COYL is the thermostat for the inside of a human life.
