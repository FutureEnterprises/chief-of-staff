# COYL Outreach Templates

> Paste-ready outreach templates for the founder. Each file is one
> outreach move from the May 2026 strategy stack. Edit the bracketed
> placeholders, then send.

## How to use

1. Open the file. Read it once. The voice is locked.
2. Edit only the `[BRACKETED]` placeholders. Don't drift the body.
3. Send. Don't agonize.

The voice across all templates: founder-first, specific, no fluff, no
exclusivity, no desperation. Every template ends with a soft hook, not
a hard ask. We are not pitching to sell — we are establishing context.
