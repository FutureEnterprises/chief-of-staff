### Three-paragraph description

Microsoft Viva measures focus time, meeting load, sentiment, and
wellbeing trends across the enterprise — but reporting is not
intervention. Employees see the data and still drop the follow-up,
skip the deep-work block, send the reactive Slack. COYL fires in the
three-second window before that happens.

COYL learns each user's danger windows (the recurring time blocks
when their autopilot script runs), classifies their excuse patterns
(the sentence they tell themselves before they drift), and delivers
precision interrupts via Teams DM + Edge browser extension at the
exact moment intervention changes the outcome. The result is the
behavior layer Viva has been missing for four years.

For Microsoft 365 customers: zero new tools to install, zero new
data silos. COYL reads commitment signals from Teams meetings,
calendar entries, and chat (with explicit consent), fires interrupts
back through the channels users already inhabit. We are seeking
partnership status to deploy COYL inside three Microsoft 365 enterprise
environments as a 30-day instrumented pilot. We fund the pilot. You
provide the customer relationship. We co-author the outcome data.

### Target customer
Mid-market and enterprise Microsoft 365 customers (1,000-50,000 seats)
with active Viva Insights deployments and measurable focus-time gaps.

### Technical integration
- Microsoft Teams Bot Framework (Teams DM delivery)
- Microsoft Graph API (calendar + commitment-signal extraction)
- Edge browser extension (Manifest V3, sideload + Edge Add-on store)
- SCIM provisioning via Azure AD
- SOC 2 in progress; HIPAA-eligible BAA available for healthcare verticals
- Data residency: customer-tenant-isolated; no cross-customer training

### Why now
The behavioral interrupt protocol (BIP v0.1, Apache 2.0) is open-spec
at coyl.ai/protocol. Microsoft customers are the largest enterprise
cohort where Viva is already collecting the signal but missing the
intervention layer. COYL closes the gap.
