# Monthly Investor Update — Template

> Send once a month, same day each month, no exceptions. Even when the
> month was bad. Especially when the month was bad. The cadence is the
> signal. Per the $6B memo: the seed round is the SIGNAL, not the cash.
> Treat each update as if a strategic acquirer might read it.

`COYL — investor update — [Month YYYY] — [one-line headline]`

## Discipline notes (don't ship this section)

1. Same format every month. Investors learn the shape; they read it
   faster; trust compounds.
2. Real numbers. Never vanity metrics. If the metric goes down, say
   so and explain why.
3. The ask is ALWAYS specific. "Help" is not an ask. "Intro to
   [name]" is an ask.
4. Send same day each month. Cadence is a signal.
5. If a strategic acquirer's investor sees this update via a portfolio
   forward, the update should read as "this company is real" without
   any other context. Write for that reader.
