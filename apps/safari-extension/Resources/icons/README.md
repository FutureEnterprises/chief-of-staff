# Icons — placeholder

Drop in three PNGs before submission:

- `icon-16.png` (16×16)
- `icon-48.png` (48×48)
- `icon-128.png` (128×128)

Match the Chrome build's icon set — `apps/extension/icons/` — so the
COYL brand mark is identical across stores. For Safari/macOS the system
also accepts higher-resolution variants (`icon-512.png`) but they're
not required by the manifest.
